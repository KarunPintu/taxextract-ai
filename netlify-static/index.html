TaxExtract AI

Local-First Intelligent Document Processing

Netlify build fixed: this repository is a Python Streamlit application, not a Hugo static site. Netlify cannot run the Streamlit server, so this static page is published only to avoid the incorrect Hugo build command.
Recommended Host Deploy the real app on Streamlit Community Cloud and select app.py as the entry file.
Local Run Install requirements and start the app with: streamlit run app.py
Other Server Hosts Render, Railway, or Heroku can run Streamlit because they support long-running Python web processes.

Netlify is excellent for static websites. TaxExtract AI needs a live Python process for uploads, OCR/parsing, session state, review workflow, and exports.
